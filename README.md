# Zigbee
[Driver for Zibee 3.0 USB Dongle Plus](https://dongle.sonoff.tech/sonoff-dongle-flasher/)

Pour utiliser l'IDE Arduino 2.3.4, il faudras Ajouter le lien officiel : * Ouvre ton Arduino IDE.

Va dans Fichier > Préférences.

Tout en bas, cherche la case URL de gestionnaire de cartes supplémentaires.

Colle cette adresse exacte :
https://siliconlabs.github.io/arduino/package_silabs_index.json

Clique sur OK.

Installer le moteur "Silicon Labs" :

Va dans le menu de gauche et clique sur l'icône du Gestionnaire de cartes (la petite icône en forme de puce avec une carte réseau, juste en dessous du gestionnaire de bibliothèques).

Cherche "Silicon Labs".

Installe le paquet qui s'affiche.

Sélectionner ta carte :

Une fois l'installation terminée, branche ton Arduino Nano Matter.

Dans le menu déroulant en haut de ton écran (là où tu sélectionnais ton ancien Nano), tu pourras maintenant chercher et sélectionner Arduino Nano Matter.
